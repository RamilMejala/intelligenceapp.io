VIRTUALIZATION

Virtualization refers to the creation of a virtual (rather than actual) version of something, including virtual computer hardware platforms, operating systems, storage devices, and computer network resources. It is a proven software technology that makes it possible to run multiple operating systems and applications on the same server at the same time.
